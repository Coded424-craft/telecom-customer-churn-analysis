# Telecom Customer Churn Analysis Dashboard

## Overview

This project analyzes customer churn behavior in a telecommunications company using Microsoft Excel.

## Objective

Identify factors associated with customer churn and provide actionable retention insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Dashboard Design

## Key Findings
1. Month-to-month customers experience the highest churn.
2. Fiber optic customers churn more frequently than DSL customers.
3. Electronic check users show significantly higher churn.
4. Churn decreases as customer tenure increases.

## Dashboard Features
- Interactive Filters
- KPI Cards
- Churn Analysis Charts
- Key Business Insights
